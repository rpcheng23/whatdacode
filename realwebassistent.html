Now you might be wondering which programming languge to learn first. This actually depends on what you want to do.

If you want to do web development I recommemd that you learn HTML, CSS and Javascript.They are all quite easy to learn. The webpage you are looking at is written in HTML5, CSS3 and Javascript.

If you are intersted in data science, data analysis, AI, machine learning you should learn Python.It is easy to learn but it can be scaled up to great heights.It is a very powerful programming languge, most of Facebook, Youtube, Google and Reddit is made out of Python. Python can also be used for web development using frameworks like django and flask.

If you are intersted in iphone app development you should learn Swift

If you are intersted in android app development you should learn Java/Kotlin.

For gaming you should learn C++ but I do not recommemd learning C++ as your first programming languge as it is very diffcult to learn and understand.
